# Rendering Grass Swaying in the Wind

This application renders dynamic grass that can be influenced by varying types of wind that can be configured by the user. 

This project was created as part of the *Graphics Programming* course at the *IT University of Copenhagen* as part of the *Master in Games*.

This project was graded with a 12/12 (or an A on the ECTS scale).

The repository includes the report that was handed in to accompany the source code. 

More information on the project can be found here: https://www.romigeleijn.com/portfolio/rendering-dynamic-grass/
